# RAG (Retrieval-Augmented Generation) Systems

Comprehensive collection of resources, methods, theories, and implementations for RAG systems.

## Directory Structure

### `/resources`
External documentation, papers, and reference materials.

### `/projects`
Git submodules and standalone RAG projects.

### `/documentation`
Internal documentation and guides.

### `/methods`
Different RAG methodologies and approaches:
- Naive RAG
- Advanced RAG
- Modular RAG
- Graph RAG
- Agentic RAG

### `/theories`
Theoretical foundations:
- Information retrieval theory
- Semantic similarity
- Dense vs. sparse retrieval
- Hybrid approaches

### `/types`
Classification of RAG system types:
- Document-based RAG
- Knowledge graph RAG
- Multi-modal RAG
- Conversational RAG

### `/systems`
Complete RAG system implementations and architectures.

### `/parsing`
Document and file parsing:
- `/documents` - PDF, DOCX, HTML, Markdown parsers
- `/files` - Code files, structured data parsers

### `/import-export`
Data ingestion and export pipelines.

### `/chunking`
Text chunking strategies:
- `/strategies` - Chunking algorithms and approaches
- `/implementations` - Concrete implementations

### `/embeddings`
Embedding models and vector representations.

### `/retrieval`
Retrieval algorithms and ranking methods.

### `/indexing`
Vector database indexing strategies.

## Key Concepts

- **Chunking**: Breaking documents into retrievable segments
- **Embedding**: Converting text to vector representations
- **Retrieval**: Finding relevant chunks for a query
- **Augmentation**: Enhancing LLM prompts with retrieved context
- **Indexing**: Efficient storage and lookup of embeddings
