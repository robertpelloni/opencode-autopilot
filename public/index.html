Consensus Modes

  • Simple Majority: Requires >50% approval.
  • Supermajority: Requires >66% approval.
  • Unanimous: Requires 100% approval.
  • Weighted: Uses supervisor weights and confidence scores.
  • CEO Override: Lead supervisor decides.
  • CEO Veto: Lead supervisor can veto decisions.
  • Hybrid CEO-Majority: CEO breaks ties, otherwise majority.
  • Ranked Choice: Supervisors rank options.

CLI Tools

The system automatically detects installed CLI tools. Supported tools include OpenCode, Claude, Aider, Cursor, and many others.

Usage

Start a new session to begin. The council will monitor your activity and provide guidance based on the configured consensus rules.
