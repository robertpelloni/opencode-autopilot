Consensus Modes

  • Simple Majority: Requires >50% approval.
  • Supermajority: Requires >66% approval.
  • Unanimous: Requires 100% approval.
  • Weighted: Uses supervisor weights and confidence scores.
  • CEO Override: Lead supervisor decides.
  • CEO Veto: Lead supervisor can veto decisions.
  • Hybrid CEO-Majority: CEO breaks ties, otherwise majority.
  • Ranked Choice: Supervisors rank options.

Advanced Features

Collaborative Debates: Create invites for human team members to join debates alongside AI supervisors.

Debate Simulator: Replay past debates or run simulations to test different team compositions and consensus modes.

Analytics: View detailed performance metrics and visual trends in the Analytics Dashboard.

Veto & Notifications: Review and approve sensitive decisions made by the council. Use the bell icon to manage pending veto requests.

Usage & Limits: Monitor API usage, costs, and set specific rate limits for each provider to control budget.

Workspaces: Create and switch between isolated workspaces to manage different projects or contexts.

Debate Templates: Use predefined or custom templates to quickly start new debates with specific configurations.

Fine-Tuning: Upload datasets and launch fine-tuning jobs to improve model performance on specific tasks.
